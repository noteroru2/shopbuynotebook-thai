# รายงานการอัปเกรดคุณภาพเนื้อหา — Batch 3 (GSC Opportunity Upgrade)

**โปรเจกต์:** เว็บร้านรับซื้อโน๊ตบุ๊ค.com
**เป้าหมาย:** เลือกและอัปเกรด 60 URLs ที่มีอันดับเฉลี่ยช่วง 11–80 เพื่อผลักดันสู่หน้าแรกของ Google

---

## 1. ผลลัพธ์โดยย่อและสรุปสถานะล่าสุด
- **สตรีมไลน์หน้า SEO:** จำนวน sitemap ทั้งหมดอยู่ที่ **372 URLs** (ไม่เพิ่มหน้าใหม่ และไม่อัพเดท URL slugs)
- **Tier C & Combo Pages:** ไม่มีหน้า Combo เกิน 2 ชั้นใน sitemap และหน้าประเภท Tier C ถูกตั้งค่าเป็น `noindex, nofollow` ครบถ้วนตามมาตรฐาน
- ** duplicate_titles**: ได้รับการปรับแต่งจนมีค่าเท่ากับ **0** (ไม่มีหน้าใดใน dist มี title ซ้ำกัน)
- **การผ่านการตรวจสอบ (QA & Rebuild):** Rebuild ผ่านสมบูรณ์ (`npm run build` สำเร็จ ทั้งหมด) และรันสคริปต์ `seo-qa.py` ผลคือ **ALL SMOKE TESTS PASSED**

---

## 2. กลุ่มหน้าและจำนวนที่ได้รับการอัปเกรดคุณภาพ (รวม 60 URLs)
ตามที่กำหนดในแผนงาน `batch-3-url-selection.md` เราแบ่งหน้าออกเป็น 4 กลุ่มหลัก:
- **Group A (20 URLs):** หน้าแลนดิ้งยอดนิยม (Quick-win / High-opportunity) รวมถึง Astro Pages 7 หน้า และ Brand/Series หน้าหลัก 13 หน้า
- **Group B (20 URLs):** หน้าโอกาสของแบรนด์และซีรีส์/โมเดลเฉพาะเจาะจง (Acer Aspire 5, Asus ROG, Dell XPS, HP Omen ฯลฯ)
- **Group C (10 URLs):** หน้าที่มีปัญหา CTR (แล็ปท็อปชำรุด อาการเสีย และบทความประกอบ)
- **Group D (10 URLs):** หน้าที่มีปัญหา Cannibalization ข้อมูลกินกันเอง (แบรนด์ vs บทความราคาเท่าไหร่)

## 3. การดำเนินการตามข้อจำกัดพิเศษ (Constraints Compliance)
- **ข้อจำกัด MDM / BIOS Lock:**
  - ไม่มีข้อมูลการแนะนำวิธีการเลี่ยงระบบ (Bypass) ในหน้าแบรนด์/เงื่อนไขที่ประเมิน
  - ใช้ถ้อยคำแบบปลอดภัย (Safe Wording): อธิบายเป็นเครื่องที่มีข้อจำกัดการใช้งาน (Restricted Device) และจะประเมินราคาตามจริงของชิ้นส่วนอะไหล่ที่สามารถนำมาใช้งานต่อได้และเอกสารยืนยันสิทธิ์
- **ข้อจำกัดการลบข้อมูล (Data Wiping):**
  - หลีกเลี่ยงการรับประกันลบข้อมูล ทั้งหมด หรือระดับองค์กรแบบถาวร
  - ปรับปรุงเนื้อหาให้คำแนะนำที่ปลอดภัย: แนะนำให้เจ้าของเครื่องทำการสำรองข้อมูลส่วนบุคคล (Backup), ออกจากระบบบัญชีสำคัญ (Sign out Apple ID / iCloud / Microsoft Account) และดำเนินการรีเซ็ตผ่านเมนูมาตรฐานของระบบปฏิบัติการ
- **ข้อจำกัด MacBook Pro M5:**
  - หน้า `macbook-pro-m5.md` ไม่มีข้อมูลคาดเดาสเปก ปีเปิดตัว หรือชิปพิเศษ
  - ให้เพียงแนวทางประเมินตามประวัติและเกณฑ์ทั่วไป (ความจุ RAM/SSD, สุขภาพแบต, อุปกรณ์เสริม, สิทธิ์บัญชี และสภาพภายนอก)
- **คีย์เวิร์ดโฆษณาเกินจริง (Banned Claims Cleanup):**
  - ทำการตรวจสอบและเปลี่ยนข้อความใน 60 หน้าเป้าหมายทั้งหมดให้ปลอดคำต้องห้าม (เช่น *อย่างรวดเร็ว, แน่นอน, แน่นอน, ดีที่สุด, เต็มประสิทธิภาพ, ราคาประเมินอ้างอิง, ราคาประเมินอ้างอิง, หลายรุ่นยอดนิยม, หลากหลายอาการ, เร็วขึ้นเมื่อข้อมูลครบ, ติดต่อผ่าน LINE/โทรตามช่องทางที่ระบุ*) โดยไม่มีสถิติคำเหล่านั้นหลงเหลืออยู่ใน 60 หน้านี้เลย

## 4. วิธีการสุ่มและตัวอย่างคู่โครงสร้างลิงก์ (Internal Links Structure)
การเชื่อมโยงข้ามหน้าได้รับการวางแผนอย่างเป็นระบบ โดยแต่ละหน้าจะมี:
1. ลิงก์ตรงกลับไปยัง **Money Page** ที่เกี่ยวข้อง
2. ลิงก์ข้ามไปยังหน้าซีรีส์/แบรนด์ย่อย 1-3 ลิงก์
3. ลิงก์ไปยังอาการเสีย/บทความประกอบ 1-2 ลิงก์ เพื่อเป็นประโยชน์ในการหาข้อมูล

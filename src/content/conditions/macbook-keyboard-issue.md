## แนวทางการรับซื้อและประเมินราคา

บริการรับซื้อและประเมินราคา MacBook คีย์บอร์ดเสีย ปุ่มกดไม่ทำงานบางปุ่ม แป้นพิมพ์ขึ้นเบิ้ล หรือมีปัญหาปุ่มหลุดหลวมแตกหัก ช่างจะประเมินราคาพิจารณาจากชิปประมวลผล ขนาดของเครื่อง ความจุ RAM/SSD และหักลบราคาค่าซ่อมอะไหล่แป้นพิมพ์ตามสเปกรุ่นย่อย หากท่านประสงค์จะขายเครื่อง แนะนำให้ถ่ายรูปจุดที่มีปุ่มหลุด และแคปหน้าจอ About This Mac เพื่อส่งให้ช่างคำนวณราคาประเมินล่วงหน้าครับ

### ข้อมูลจำเพาะและจุดประเมินสำหรับ MacBook คีย์บอร์ดเสีย
- **ประเภทของแป้นพิมพ์ MacBook ที่พบปัญหาย่อย:**
  - *Butterfly Keyboard (รุ่นปี 2015-2019):* แป้นพิมพ์ผีเสื้อที่ปุ่มคีย์บอร์ดค้างบ่อยเมื่อมีเศษฝุ่นสะสมใต้แป้น
  - *Magic Keyboard (รุ่นปี 2020 ขึ้นไป):* แป้นพิมพ์กลไกขากรรไกร (Scissor Switch) ที่ทนทานกว่า
- **คำแนะนำ:** หากแป้นพิมพ์เสียบางปุ่ม แนะนำให้เชื่อมต่อคีย์บอร์ดบลูทูธภายนอกเพื่อล็อกเอ้าท์ออกจากระบบ Apple ID ให้เรียบร้อยก่อนนำส่ง

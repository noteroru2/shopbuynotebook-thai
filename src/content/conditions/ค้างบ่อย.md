## รับซื้อโน๊ตบุ๊คค้างบ่อย / ช้า

อาการค้างหรือช้ามักมาจาก SSD เต็ม RAM ไม่พอ ฝุ่นในพัดลม หรือแบตเสื่อม — ไม่ใช่ “เสียทั้งเครื่อง” เสมอไป

### อาการนี้ขายได้ไหม

ขายได้ โดยเฉพาะเครื่องที่ยังเปิดติดและสเปกยังต้องการในตลาด ราคาปรับตามสาเหตุที่แจ้งและสภาพจริง

### ประเมินจากอะไร

- สเปก CPU RAM SSD/HDD
- อาการ: ค้างตอนเปิดเครื่อง / ใช้ Chrome / เล่นเกม / ชาร์จอยู่
- ความร้อนและเสียงพัดลม
- อายุเครื่องโดยประมาณ

### ควรซ่อมก่อนขายไหม

อัปเกรด SSD หรือเพิ่ม RAM อาจช่วยได้ถ้าคุ้มค่า แต่ถ้าซ่อมแพงกว่ามูลค่าที่ขายได้ แนะนำขายตามสภาพและแจ้งอาการตรง ๆ

### ข้อมูล/รูปที่ควรส่ง

- แคป Task Manager (CPU RAM Disk)
- แจ้งว่าค้างบ่อยแค่ไหน และทำอะไรอยู่
- รูปพัดลม/ช่องระบาย (ถ้าร้อนจัด)
- รูปตัวเครื่องและสติ๊กเกอร์รุ่น

### รุ่นที่ยังมีโอกาสขายได้

- สายงาน SSD 256GB+ ที่ค้างเพราะ RAM 4GB (ยังมีตลาดหลังอัปเกรด)
- Gaming ที่ค้างเพราะความร้อน แต่ GPU ยังดี
- MacBook แบตเสื่อมทำให้ throttle (แจ้ง Battery Health)

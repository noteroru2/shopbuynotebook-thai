โน๊ตบุ๊คที่ขึ้น **จอฟ้า** ยังส่งประเมินได้ครับ ร้านจะดูว่าปัญหาเกิดจากซอฟต์แวร์, SSD/HDD, RAM หรือมีแนวโน้มไปถึงเมนบอร์ดร่วมด้วย

ถ้ามีรูป error code หรือวิดีโออาการตอนเครื่องค้างก่อนจอฟ้า จะช่วยให้ประเมินใกล้เคียงสภาพจริงมากขึ้น

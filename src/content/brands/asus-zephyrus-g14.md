## ร้านรับซื้อ Zephyrus G14 ดูอะไรบ้าง

Zephyrus G14 เป็นเกมมิ่งโน๊ตบุ๊คที่คนขายมักค้นตรงรุ่น เพราะจุดเด่นอยู่ที่ความบางเบาแต่ยังได้ GPU แยก ร้านจะดูทั้งสเปกการ์ดจอ หน้าจอ ความร้อน และสภาพบอดี้อย่างละเอียดก่อนประเมิน

## ส่งอะไรให้ประเมินเร็ว

- รูปตัวเครื่องรอบด้านและหน้าจอ
- รูปสเปก CPU / RAM / SSD / GPU
- รูปอะแดปเตอร์เดิม
- แจ้งอาการเรื่องความร้อนหรือพัดลมดังถ้ามี
- แจ้งว่ามีการอัปเกรด RAM / SSD หรือไม่

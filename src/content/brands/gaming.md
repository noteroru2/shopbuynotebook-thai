### ปัจจัยที่ส่งผลต่อราคารับซื้อเกมมิ่งโน๊ตบุ๊ค

**GPU และ CPU** — รุ่น RTX 30/40 Series หรือ AMD รุ่นใหม่ ๆ มักมีตลาดรองรับดีกว่า GPU รุ่นเก่า ควรแคปหน้าจอ Device Manager หรือ GPU-Z / หน้าสเปกในเกมเพื่อยืนยันรุ่นจริง

**RAM และ SSD** — 16GB RAM และ SSD NVMe 512GB ขึ้นไป มักได้รับการประเมินดีกว่าสเปกต่ำ หากอัปเกรด RAM/SSD เอง แจ้งมาได้

**หน้าจอ** — รอยแตก dead pixel หรือ backlight bleed มีผลต่อราคา จอ 144Hz ขึ้นไปหรือ QHD มักเป็นจุดขายของเครื่องเกมมิ่ง

**ระบบระบายความร้อน** — พัดลมดังผิดปกติ เครื่องร้อนจัด หรือดับเอง ควรแจ้งตรง ๆ ไม่ใช่ปัญหาเดียวที่ทำให้ขายไม่ได้ แต่มีผลต่อกรอบราคา

**แบตและตัวเครื่อง** — แบตเสื่อม รอยบุบ บานพับหลวม คีย์บอร์ด/RGB ที่เสีย ล้วนใช้ประกอบการประเมิน

### เตรียมเครื่องก่อนส่งประเมิน

1. สำรองข้อมูลเกมและไฟล์งาน  
2. Sign out บัญชี Steam/Epic/Battle.net และ cloud ที่ผูกไว้  
3. ทำ Factory Reset หากสะดวก (แนะนำก่อนส่งมอบ)  
4. ถ่ายรูปตัวเครื่อง หน้าจอ คีย์บอร์ด และแคปสเปก  
5. รวบรวมที่ชาร์จ กล่อง ใบเสร็จ (ถ้ามี)

### โน๊ตบุ๊คเกมมิ่งมือสอง vs ใหม่

เครื่องมือสองมักมีรอยใช้งานและแบตที่เริ่มเสื่อม แต่สเปก GPU ยังเป็นตัวกำหนดราคาหลัก
การส่งแคปสเปกและรูปสภาพจริงจะได้กรอบราคาที่สะท้อนตลาดมือสอง ไม่ใช่ราคาเครื่องใหม่
หากเครื่องมีอาการด้านความร้อน แจ้งตรง ๆ จะช่วยให้ประเมินได้ใกล้เคียงหน้างาน

### อัปเกรด RAM SSD มีผลอย่างไร

หากอัปเกรด RAM หรือ SSD เอง แจ้งความจุและยี่ห้อที่ใช้ มักช่วยดึงราคาได้ดีกว่าเครื่องสเปกเดิมจากโรงงาน ส่งแคปหน้าจอ Task Manager หรือ CrystalDiskInfo ถ้ามี การอัปเกรดที่ทำอย่างถูกต้องมักเป็นจุดขายของเครื่องมือสอง

### ขายเพื่ออัปเกรดรุ่นใหม่

หลายคนขายเกมมิ่งโน๊ตบุ๊ครุ่นเก่าเพื่อสมทบทุน RTX รุ่นใหม่ การส่งสเปก GPU เดิมและสภาพจริงจะได้กรอบราคาที่ช่วยวางแผนงบได้ ไม่จำเป็นต้องรู้ราคาเป๊ะก่อนส่ง — ขอประเมินเบื้องต้นฟรีได้

### ตลาดมือสอง vs ขายเอง

การขายเองอาจได้ราคาดีกว่าในบางเคส แต่ใช้เวลาและมีความเสี่ยงจากผู้ซื้อ การขายให้ร้านรับซื้อแลกกับความรวดเร็วและความชัดเจน หากต้องการเงินเร็วหรือไม่อยากนัดหลายรอบ การประเมินจากรูปและสเปกเป็นทางลัดที่เหมาะ

### จอ 144Hz QHD และ OLED มีผลอย่างไร

จอความถี่สูงหรือความละเอียดพิเศษมักเป็นจุดขายของเกมมิ่งโน๊ตบุ๊ค หากจอมีรอยแตกหรือ dead pixel แจ้งตำแหน่งและขนาด จอ OLED ที่มี burn-in ควรแจ้งและส่งรูปเมื่อแสดงภาพนิ่ง ราคาปรับตามสภาพจอและต้นทุนทดแทน

### ที่ชาร์จ Watt และอุปกรณ์เสริม

เกมมิ่งโน๊ตบุ๊คหลายรุ่นใช้ที่ชาร์จ Watt สูง หากไม่มีที่ชาร์จเดิม แจ้งล่วงหน้า ราคาอาจปรับตามความครบของชุด ไม่ใช่เหตุผลที่ปฏิเสธการประเมิน หากมี cooling pad หรืออุปกรณ์เสริม แจ้งได้แต่ไม่จำเป็นต้องมีครบ

### เทียบกับเครื่องออฟฟิศและ MacBook

หากมีทั้งเกมมิ่งและออฟฟิศในล็อต แยกรายการประเมิน เกมมิ่งดู GPU เป็นหลัก ออฟฟิศดู CPU RAM และสภาพใช้งาน หน้า [เช็คราคาโน๊ตบุ๊คมือสอง](/เช็คราคาโน๊ตบุ๊คมือสอง/) เหมาะกับเครื่องทั่วไป หน้านี้เน้น GPU และระบบความร้อน

### NVIDIA vs AMD ในการประเมิน

การ์ดจอ NVIDIA GeForce RTX มักมีตลาดมือสองกว้างกว่าในหลายช่วง AMD Radeon บางรุ่นมีตลาดเฉพาะ แคปหน้าจอ Device Manager หรือ GPU-Z ช่วยยืนยันรุ่นจริง ไม่ใช่แค่ชื่อรุ่นโน๊ตบุ๊ค หาก GPU ถูก disable หรือ driver มีปัญหา แจ้งและส่งรูป error ถ้ามี

### เครื่องเกมมิ่งที่ใช้งานหนัก

### เกมมิ่งโน๊ตบุ๊คในองค์กร

### ทำความเข้าใจ TGP และสเปกการ์ดจอ

TGP (Total Graphics Power) คือกำลังไฟที่ GPU ใช้ได้จริงในเครื่อง รุ่นเดียวกันอาจมี TGP ต่างกัน ส่งแคปสเปกจาก GPU-Z หรือหน้าจอ NVIDIA Control Panel ช่วยยืนยัน หาก GPU ถูก disable ใน BIOS แจ้งด้วย

### จอ G-Sync FreeSync และความถี่สูง

จอ 144Hz 165Hz หรือ 240Hz มักเป็นจุดขายของเกมมิ่ง หากจอมีอาการ ghosting หรือ backlight bleed แจ้งและส่งรูป ราคาปรับตามสภาพจอ ไม่ใช่แค่สเปกบนกระดาษ

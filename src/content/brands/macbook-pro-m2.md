## ข้อมูลสเปกหลักของรุ่น MacBook Pro M2 ที่มีผลต่อราคาประเมิน

**MacBook Pro M2** (ขนาด 13.3 นิ้ว เปิดตัวปี 2022) เป็นรุ่นอัปเกรดสเปกโดยใช้ชิป **Apple Silicon M2** (ประกอบด้วย CPU 8-core และ GPU 10-core) ในตัวเครื่องดีไซน์คลาสสิก โดดเด่นด้วยการมีพัดลมระบายความร้อน, แผงควบคุมสัมผัส Touch Bar, และแบตเตอรี่ที่ใช้งานยาวนานเป็นพิเศษ ในตลาดมือสอง รุ่นนี้เป็นที่ต้องการอย่างมากของกลุ่มฟรีแลนซ์, โปรแกรมเมอร์, และคนทำงานที่ต้องการเครื่องที่ไว้ใจได้ในระยะยาว

สเปกย่อยที่ผู้ขายควรแจ้งเพื่อการประเมินราคา:
-   **หน่วยประมวลผล (ชิป):** Apple M2 (GPU 10-core)
-   **หน่วยความจำ (RAM):** 8GB / 16GB / 24GB Unified Memory (RAM ฝังบอร์ด ไม่สามารถอัปเกรดภายหลังได้)
-   **พื้นที่เก็บข้อมูล (SSD):** 256GB / 512GB / 1TB / 2TB
-   **หน้าจอแสดงผล:** หน้าจอ Retina ขนาด 13.3 นิ้ว สัดส่วน 16:10 ความละเอียด 2560 x 1600 พร้อมรองรับช่วงสีกว้าง P3

## จุดที่ควรตรวจสอบเฉพาะรุ่น MacBook Pro M2

เพื่อความรวดเร็วและแม่นยำในการเสนอราคาประเมินอ้างอิง แนะนำให้ผู้ขายตรวจสอบจุดสำคัญดังต่อไปนี้:

-   **ระบบแผงควบคุมสัมผัส Touch Bar:** ตรวจดูว่าแผงไฟ Touch Bar สว่างสม่ำเสมอดี และระบบสัมผัสทำงานตอบสนองได้ปกติทุกฟังก์ชัน
-   **ระดับสุขภาพแบตเตอรี่ (Battery Health) และรอบชาร์จ (Cycle Count):** เช็คระดับสุขภาพแบตเตอรี่ หากต่ำกว่า 80% หรือขึ้นแจ้งเตือนแนะนำให้เข้ารับบริการ (Service)
-   **สภาพบอดี้ภายนอก:** ตัวเครื่องอลูมิเนียม ตรวจเช็คว่าไม่มีรอยร้าว รอยยุบตัว หรือรอยตกกระแทกตามมุมเครื่องทั้ง 4 ด้านอย่างละเอียด

## อาการเสื่อมและอาการชำรุดที่พบบ่อยที่มีผลต่อการปรับราคารับซื้อ

-   **อาการแบตเตอรี่เสื่อม:** แบตเตอรี่เริ่มเก็บไฟได้น้อยลงตามรอบการใช้งานปกติ (สามารถเช็คอาการและส่งประเมินเพิ่มได้ที่หน้า [รับซื้อโน๊ตบุ๊คแบตเสื่อม](/รับซื้อโน๊ตบุ๊ค/แบตเสื่อม/))
-   **ปุ่มคีย์บอร์ดและพัดลมระบายความร้อน:** ตรวจสอบปุ่มกดคีย์บอร์ดกลไก Scissor Switch ว่าทำงานปกติ และพัดลมระบายความร้อนหมุนปกติ ไม่มีเสียงสั่นสะเทือน

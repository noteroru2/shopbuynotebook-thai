เวลาจะส่งเครื่องให้ร้านรับซื้อหรืออยาก **เช็คราคาโน๊ตบุ๊ค** ก่อนขาย หลายคนอาจจะติดขัดตรงที่ “ไม่แน่ใจว่าโน๊ตบุ๊คของตัวเองคือรุ่นอะไร” ความจริงแล้วคุณไม่จำเป็นต้องจำชื่อรุ่นให้ถูกต้องทั้งหมดตั้งแต่แรก เพียงแค่ถ่ายรูปหรือดึงข้อมูลสำคัญบางจุดส่งให้ทางร้าน ช่างเทคนิคก็สามารถช่วยตรวจสอบรุ่นย่อยและประเมินราคาประเมินอ้างอิงเบื้องต้นให้ได้อย่างรวดเร็ว

บทความนี้สรุปขั้นตอนและวิธีดู **รุ่นโน๊ตบุ๊ค Windows** แบบง่าย ๆ ที่สามารถทำได้ด้วยตัวเองภายในเวลาไม่กี่นาที เพื่อช่วยให้การประเมินราคาผ่านทาง LINE ของคุณสะดวกและได้รับกรอบราคาที่ใกล้เคียงกับความเป็นจริงมากที่สุด

> ลิงก์สำคัญ: หากต้องการส่งข้อมูลประเมินราคาอย่างรวดเร็ว สามารถติดต่อผ่านหน้า [รับซื้อโน๊ตบุ๊ค](/) หรือเช็ครายละเอียดบริการหลักที่หน้า [เช็คราคาโน๊ตบุ๊ค](/เช็คราคาโน๊ตบุ๊ค/)

## วิธีตรวจสอบรุ่นโน๊ตบุ๊ค Windows จากภายในระบบปฏิบัติการ

หากตัวเครื่องของคุณยังคงใช้งานได้และเปิดหน้าจอติดปกติ วิธีที่แม่นยำที่สุดคือการดูผ่านเมนูระบบของ Windows ซึ่งมีขั้นตอนดังนี้:

### วิธีที่ 1: ดูผ่านเมนู Settings (แนะนำสำหรับผู้ใช้ทั่วไป)
1. คลิกที่ปุ่ม **Start** (โลโก้ Windows) บริเวณมุมซ้ายล่าง หรือกดปุ่ม Windows บนแป้นพิมพ์
2. คลิกเลือกไอคอนรูปเฟืองเพื่อเข้าสู่ **Settings**
3. คลิกเลือกหัวข้อ **System** แล้วเลือกเมนู **About** ที่อยู่ด้านล่างสุด
4. ถ่ายรูปหน้าจอในส่วนของ **Device specifications** ส่งมาทาง LINE ซึ่งจะระบุข้อมูล CPU (Processor) และขนาดความจุ RAM ที่ติดตั้งอยู่จริง

### วิธีที่ 2: ดูผ่านหน้าต่าง System Information (ข้อมูลละเอียด)
1. กดปุ่ม Windows บนคีย์บอร์ดพร้อมพิมพ์คำว่า `msinfo32` ในช่องค้นหาแล้วกด Enter
2. หน้าต่าง **System Information** จะแสดงข้อมูลระบบทั้งหมดขึ้นมา
3. มองหาบรรทัดที่เขียนว่า **System Model** และ **Processor** จากนั้นบันทึกภาพหน้าจอนี้เพื่อส่งให้ช่างตรวจสอบรหัสรุ่นย่อย

### วิธีที่ 3: ดูผ่านโปรแกรม DirectX Diagnostic Tool (สำหรับ Gaming Notebook)
1. กดปุ่ม Windows ร่วมกับปุ่ม R บนคีย์บอร์ด (Windows + R) เพื่อเปิดกล่องข้อความ Run
2. พิมพ์คำว่า `dxdiag` แล้วกด Enter
3. ระบบจะแสดงหน้าต่างเครื่องมือตรวจสอบขึ้นมา ถ่ายรูปแท็บ **System** (แสดง CPU/RAM) และแท็บ **Display** (แสดงรุ่นการ์ดจอแยก) ซึ่งข้อมูลชิปการ์ดจอนี้มีผลอย่างมากต่อราคาประเมินของเครื่องเล่นเกม

## วิธีตรวจสอบรุ่นโน๊ตบุ๊ค Windows จากบอดี้ภายนอก (กรณีเปิดไม่ติด)

หากเครื่องของคุณมีอาการชำรุดเปิดไม่ติด (สามารถเช็คแนวทางการขายเครื่องเสียได้ที่หน้า [รับซื้อโน๊ตบุ๊คเสีย](/รับซื้อโน๊ตบุ๊ค/เครื่องเสีย/)) คุณยังคงตรวจสอบรุ่นย่อยได้จากจุดต่าง ๆ รอบตัวเครื่อง:

*   **สติ๊กเกอร์ใต้ฐานเครื่อง (Bottom Case):** โน๊ตบุ๊คส่วนใหญ่จะมีสติ๊กเกอร์สีเงิน ดำ หรือขาว แปะอยู่ใต้เครื่อง ซึ่งจะระบุรุ่นโมเดลหลัก (Model) และเลขรหัสสินค้า (Product ID / Serial Number)
*   **บริเวณรอบแป้นพิมพ์:** แบรนด์ส่วนใหญ่มักจะพิมพ์ชื่อซีรีส์ของรุ่นไว้บริเวณมุมหน้าจอ หรือขอบด้านล่างของแป้นพิมพ์ เช่น *Asus VivoBook*, *Lenovo ThinkPad*, *Acer Nitro*
*   **กล่องบรรจุภัณฑ์และใบเสร็จรับเงิน:** หากยังคงเก็บกล่องเดิมไว้ บริเวณข้างกล่องจะมีสติ๊กเกอร์บาร์โค้ดที่ระบุสเปกและรุ่นย่อยไว้อย่างละเอียด ซึ่งเป็นข้อมูลที่ช่างสามารถนำไปใช้ตรวจสอบราคาประเมินอ้างอิงได้อย่างรวดเร็ว

## เช็คลิสต์รูปถ่ายที่แนะนำให้ส่งประเมินราคาใน LINE @webuy

เพื่อให้การประเมินราคาเบื้องต้นเป็นไปอย่างรวดเร็ว แนะนำให้ถ่ายรูปและส่งข้อมูลเหล่านี้เข้ามาในแชท:

1.  **รูปตัวเครื่องขณะกางหน้าจอใช้งาน** เพื่อแสดงสภาพความสมบูรณ์ของคีย์บอร์ดและบอดี้ฝากลาง
2.  **รูปถ่ายใต้ฐานเครื่อง** (เน้นจุดที่มีสติ๊กเกอร์บอกรุ่นและ Serial Number)
3.  **รูปถ่ายหน้าจอ About หรือสเปกเครื่อง** (กรณีเครื่องใช้งานปกติ)
4.  **รูปถ่ายอะแดปเตอร์สายชาร์จและอุปกรณ์เสริมที่มี** เช่น กระเป๋า หรือกล่องเก็บเดิม
5.  **ข้อมูลการแจ้งตำหนิหรืออาการเสื่อมตามจริง** (เช่น แบตเตอรี่เสื่อมเก็บไฟได้น้อย หรือหน้าจอมีรอยขีดข่วน) เพื่อให้กรอบราคาประเมินหน้างานไม่คลาดเคลื่อน

เมื่อจัดเตรียมรูปถ่ายเรียบร้อยแล้ว แนะนำให้ทำการสำรองข้อมูลไฟล์สำคัญและออกจากระบบบัญชีผู้ใช้งานต่าง ๆ เพื่อความปลอดภัยของข้อมูลส่วนตัวก่อนนัดส่งมอบเครื่อง โดยสามารถศึกษาขั้นตอนได้จากคู่มือ [วิธีล้างข้อมูลก่อนขายโน๊ตบุ๊ค](/blog/ก่อนขายโน๊ตบุ๊คต้องล้างข้อมูลอย่างไร/)

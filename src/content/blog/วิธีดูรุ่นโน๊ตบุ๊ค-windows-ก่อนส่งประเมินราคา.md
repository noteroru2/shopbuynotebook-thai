เวลาจะส่งเครื่องให้ร้านรับซื้อหรืออยาก **เช็คราคาโน๊ตบุ๊ค** ก่อนขาย หลายคนติดตรงเดียวคือ “ไม่รู้รุ่น” หรือ “ไม่แน่ใจว่ารุ่นไหน” ความจริงแล้วคุณไม่จำเป็นต้องจำชื่อรุ่นให้เป๊ะตั้งแต่แรก แค่ส่งข้อมูลหลัก ๆ ให้ครบ ร้านก็ช่วยไล่รุ่นและประเมินราคาเบื้องต้นให้ได้

บทความนี้สรุปวิธีดู **รุ่นโน๊ตบุ๊ค Windows** แบบเร็วที่สุด พร้อมเช็กลิสต์รูป/ข้อมูลที่ควรส่ง เพื่อให้การ **รับซื้อโน๊ตบุ๊ค** คุยกันง่ายขึ้น และคุณตัดสินใจได้เร็วขึ้นว่าจะขายหรือเก็บไว้ใช้งานต่อ

## ทำไม “รุ่น” ถึงสำคัญเวลาประเมินราคา

เวลาร้านรับซื้อประเมินราคา จะดู “กลุ่มรุ่น” และ “สเปกจริง” เช่น CPU/RAM/SSD/GPU รวมถึงสภาพแบต หน้าจอ และอุปกรณ์ที่มี รุ่นเดียวกันอาจมีหลายสเปกและราคาในตลาดมือสองต่างกันมาก ดังนั้นการรู้รุ่นหรือมีรูปที่ช่วยยืนยันรุ่น จะทำให้ประเมินไวขึ้นและลดการถามซ้ำ

## วิธีดูรุ่นโน๊ตบุ๊ค Windows แบบเร็ว (ทำได้ใน 1–2 นาที)

### 1) ดูจาก Settings (ง่ายสุด)

1. กดปุ่ม Start → ไปที่ **Settings**
2. เลือก **System**
3. เลือก **About**

ถ่ายภาพหน้าจอส่วนที่มี “Device name / Processor / Installed RAM / Device specifications” แล้วส่งให้ร้านได้เลย

### 2) ดูจาก System Information (ละเอียดขึ้น)

1. กดปุ่ม Windows แล้วพิมพ์ `msinfo32`
2. เปิด **System Information**

จุดที่ควรดู:
- **System Model**
- **System SKU** (บางเครื่องจะช่วยระบุรหัสรุ่นย่อย)
- **Processor**
- **Installed Physical Memory (RAM)**

### 3) ดูจาก DirectX Diagnostic (เช็กการ์ดจอ/จอ)

1. กด Windows แล้วพิมพ์ `dxdiag`
2. เปิดโปรแกรมแล้วดูแท็บ **System** และ **Display**

เหมาะกับเครื่องที่เป็น Gaming Notebook เพราะช่วยดู GPU ได้ชัดเจน

### 4) ดูจากสติ๊กเกอร์ใต้เครื่อง/ฝาหลัง (ใช้ได้แม้เปิดไม่ติด)

ถ้าเครื่องเปิดไม่ติด ให้พลิกดูใต้เครื่องหรือฝาหลัง มักมี:
- Model / Product name
- Serial number / S/N
- รหัสรุ่น (เช่น “A15-…”, “IdeaPad 5 …”)

ถ่ายรูปสติ๊กเกอร์ให้ชัด แล้วส่งพร้อมรูปตัวเครื่องได้เลย

## ถ้าเครื่องเปิดไม่ติด จะส่งข้อมูลยังไงให้ประเมินได้

กรณี **โน๊ตบุ๊คเปิดไม่ติด** หรือเครื่องมีอาการเสีย แนะนำส่ง:
- รูปสติ๊กเกอร์รุ่นใต้เครื่อง
- รูปตัวเครื่องรอบด้าน (ให้เห็นสภาพบอดี้/พอร์ต)
- วิดีโอสั้น ๆ ตอนกดเปิดเครื่อง (ถ่าย 5–10 วินาที)
- แจ้งว่า “มีไฟ/ไม่มีไฟ” “พัดลมหมุนไหม” “เสียงบี๊บไหม” (ถ้ามี)

การส่งข้อมูลแบบนี้ช่วยให้ร้านประเมินได้ว่ากรณีเข้าข่ายอะไร และประเมินราคาเบื้องต้นได้ใกล้เคียงขึ้น

## เช็กลิสต์รูปที่ควรส่ง (มาตรฐาน 3–5 รูป)

- หน้าเครื่องเปิดจอ (ให้เห็นสภาพจอ)
- คีย์บอร์ด/ทัชแพด (รอยใช้งาน/ปุ่ม)
- ฝาหลัง/สติ๊กเกอร์รุ่น (ยืนยันรุ่น)
- อุปกรณ์ที่มี (ที่ชาร์จ/กล่อง)
- ถ้ามีตำหนิ/อาการเสีย ให้ถ่ายเพิ่ม 1 รูปหรือวิดีโอสั้น ๆ

## รุ่น กับ รหัสรุ่น ต่างกันไหม (ทำไมร้านถึงถาม)

หลายคนสับสนคำว่า “รุ่น” กับ “รหัสรุ่น” ซึ่งทั้งสองอย่างช่วยประเมินได้ แต่มีบทบาทต่างกัน

- **รุ่น (Model/Series)**: เช่น VivoBook, ThinkPad, Nitro, Pavilion, Inspiron, Surface Laptop  
- **รหัสรุ่น (Model Number / SKU)**: เป็นชุดตัวอักษรและตัวเลขที่เจาะจงกว่านั้น เช่น A515-xx, 15ACH6, G15 5515  

เหตุผลที่ร้านถามรหัสรุ่น เพราะรุ่นเดียวกันอาจมีหลายสเปกมาก และบางรหัสรุ่นบอก “กลุ่มสเปก” ได้ใกล้เคียงกว่า

## วิธีดูรุ่นเมื่อ “เปิดเครื่องได้ แต่เข้า Windows ไม่ได้”

บางครั้งเครื่องยังติดไฟ แต่เข้า Windows ไม่ได้ หรือค้างโลโก้ ให้ลองวิธีที่ไม่ต้องเข้าเดสก์ท็อป

### 1) ดูจากหน้าจอ BIOS/UEFI

1. เปิดเครื่อง แล้วกดปุ่มเข้า BIOS (มักเป็น F2 / Del / Esc แล้วแต่ยี่ห้อ)  
2. มองหาคำว่า Product Name / System Model / Serial Number  

ถ่ายรูปหน้าจอ BIOS แล้วส่งให้ร้านได้เลย วิธีนี้ช่วยมากเวลาเครื่องบูตไม่ขึ้น

### 2) ดูจากกล่อง หรือใบเสร็จ (ถ้ายังเก็บไว้)

กล่องมักมี:
- Model / SKU  
- Serial Number  
- รายละเอียดสเปกคร่าว ๆ (บางรุ่น)  

ถ่ายรูปสติ๊กเกอร์บนกล่องส่งมาได้เลย

## วิธีส่ง “สเปก” แบบสั้น ๆ ที่คนอ่านแล้วเข้าใจทันที

เพื่อให้เช็คราคาได้เร็วขึ้น แนะนำพิมพ์สเปกเป็นบรรทัดสั้น ๆ แบบนี้ (ไม่ต้องยาว)

- รุ่น: ______  
- CPU: ______  
- RAM: ______ GB  
- SSD/HDD: ______ GB  
- การ์ดจอ: ______ (ถ้ามี)  
- สภาพ: ใช้งานปกติ / แบตเสื่อม / จอมีเส้น / เปิดไม่ติด (เลือกที่ตรง)  
- อุปกรณ์: มีที่ชาร์จ / มีกล่อง / ไม่มีที่ชาร์จ (เลือกที่ตรง)

ถ้าคุณไม่รู้ CPU/RAM/SSD ให้ส่งรูปหน้า About หรือ Task Manager ก็ได้

## วิธีดูรุ่นแบบ “ชัวร์” ด้วยคำสั่ง (สำหรับคนที่พอทำตามได้)

ถ้าคุณอยากให้ข้อมูลรุ่นแม่นขึ้นแบบไม่ต้องเดา ลองใช้คำสั่งเหล่านี้บน Windows (ทำได้ใน 1 ถึง 3 นาที) แล้วถ่ายรูปหน้าจอหรือคัดลอกข้อความส่งได้เลย

### วิธีที่ 1: Command Prompt (wmic) แบบเร็ว

1) กด Windows แล้วพิมพ์ `cmd` แล้ว Enter  
2) พิมพ์คำสั่งทีละบรรทัดนี้:

- `wmic csproduct get name`  
- `wmic csproduct get vendor`  
- `wmic bios get serialnumber`  

ถ่ายรูปหน้าจอส่วนผลลัพธ์ แล้วส่งให้ร้านได้เลย

หมายเหตุ: บางเครื่องอาจปิดการใช้งาน `wmic` ใน Windows รุ่นใหม่ ถ้าใช้ไม่ได้ให้ใช้ PowerShell แทน

### วิธีที่ 2: PowerShell (แนะนำ) ได้ข้อมูลครบกว่า

1) คลิกขวา Start → Windows Terminal (Admin ไม่จำเป็น)  
2) พิมพ์:

- `Get-CimInstance -ClassName Win32_ComputerSystem | Select-Object Manufacturer, Model`  
- `Get-CimInstance -ClassName Win32_BIOS | Select-Object SerialNumber`  

ถ้าคุณไม่อยากคัดลอก ให้ถ่ายรูปหน้าจอส่งได้เลย

### วิธีที่ 3: ดู “System SKU” (ช่วยเจาะรหัสรุ่นย่อย)

บางร้านจะถาม “SKU” เพราะระบุรหัสรุ่นย่อยได้ดีกว่า

ให้เปิด `msinfo32` แล้วมองหาบรรทัด:

- System Model  
- System SKU  

ถ่ายรูปหน้าจอส่วนนี้แล้วส่ง จะช่วยประเมินได้ใกล้เคียงขึ้นมาก

- เปิดไฟให้สว่าง หรือใช้ไฟฉายมือถือส่องเฉียง ๆ  
- วางเครื่องบนพื้นเรียบ แล้วถ่าย “ตรง ๆ” ไม่เอียง  
- ถ้าตัวหนังสือเล็ก ให้ซูม 2x แล้วถ่ายใหม่  
- ถ่าย 2 รูป: รูปกว้างเห็นตำแหน่งสติ๊กเกอร์ และรูปซูมให้เห็นตัวอักษร  

ส่วนที่ควรให้เห็น:

- Model / Product name  
- Model number (ถ้ามี)  
- Serial number (S/N) (ถ้ามี)  

## ต้องส่ง Serial Number ไหม (เรื่องความเป็นส่วนตัว)

ปกติการ “เช็คราคาเบื้องต้น” ไม่จำเป็นต้องใช้ Serial Number เสมอไป ถ้าคุณกังวลเรื่องความเป็นส่วนตัว สามารถทำได้ 2 แบบ:

- ส่งแค่ Model / SKU / รูปหน้า System ก็พอ  
- หรือถ้าจะส่ง Serial ให้ “ปิดท้าย 2 ถึง 4 ตัว” ก็ได้ เช่น XXXXX1234  

ร้านส่วนใหญ่ประเมินได้จากรุ่นและสเปกอยู่แล้ว Serial จะช่วยในบางเคสที่ต้องเช็กประกันหรือยืนยันรุ่นย่อยเท่านั้น

## ตัวอย่าง “ชุดข้อมูลมาตรฐาน” ที่ร้านอ่านแล้วประเมินได้ไว

เพื่อให้ร้านประเมินได้เร็ว แนะนำส่งเป็นชุดเดียวแบบนี้:

1) รูปหน้า About หรือ `msinfo32` (เห็น Processor, RAM, System Model)  
2) รูป Task Manager → Performance → Disk (เห็น SSD/HDD)  
3) รูปหน้าเครื่องเปิดจอ (เห็นสภาพจอ)  
4) รูปคีย์บอร์ด (เห็นรอยใช้งาน)  
5) รูปฝาหลัง/สติ๊กเกอร์รุ่น (ยืนยันรุ่น)  
6) รูปที่ชาร์จ (สำคัญ โดยเฉพาะเกมมิ่ง)  

ถ้ามีอาการเสีย ให้เพิ่ม:

- รูปตำหนิ หรือวิดีโออาการ 5 ถึง 10 วินาที

## FAQ เพิ่มเติม: คำถามเกี่ยวกับ “รุ่น” ที่คนถามบ่อยจริง

### 1) ชื่อรุ่นบนหน้าจอ กับรุ่นใต้เครื่อง ไม่ตรงกัน เกิดจากอะไร

บางครั้งหน้า Windows แสดงชื่อ “ซีรีส์” หรือชื่อเครื่องแบบกว้าง ๆ แต่ใต้เครื่องเป็น “รหัสรุ่น” ที่เจาะจงกว่า ให้ยึดรหัสรุ่นใต้เครื่องหรือ System SKU เป็นหลัก และแนบรูปประกอบ จะช่วยให้ประเมินใกล้เคียงขึ้น

### 2) ส่งแค่รูปเครื่องอย่างเดียวได้ไหม

ได้ครับ แต่จะประเมินได้กว้างและต้องถามเพิ่ม แนะนำอย่างน้อยส่ง “รูปสติ๊กเกอร์รุ่น” หรือ “รูปหน้า About” เพิ่มอีก 1 รูป จะเร็วขึ้นมาก

### 3) ถ้าเครื่องเปิดไม่ติด จะให้ข้อมูลรุ่นยังไงดีที่สุด

ส่งรูปสติ๊กเกอร์ใต้เครื่อง + รูปสภาพเครื่อง + วิดีโออาการตอนกดเปิดเครื่อง แล้วร้านจะประเมินตามสภาพให้ก่อน

อ่านต่อ: [โน๊ตบุ๊คเปิดไม่ติดขายได้ไหม](/blog/โน๊ตบุ๊คเปิดไม่ติดขายได้ไหม/)

### 4) ถ้าไม่แน่ใจว่าเป็น Notebook หรือ Ultrabook หรือ 2-in-1 ต้องบอกไหม

ไม่ต้องก็ได้ครับ แค่ส่งรุ่นและรูปสภาพจริง ร้านดูจากรุ่นได้ แต่ถ้าเป็นเครื่องจอพับ 360 องศา หรือถอดคีย์บอร์ดได้ แนะนำบอกเพิ่ม เพราะมีผลกับกลุ่มราคาในบางรุ่น

### 5) รู้รุ่นแล้ว ต้องเช็กสเปกไหม

ควรเช็กครับ เพราะรุ่นเดียวกันอาจมีหลายสเปกและราคาต่างกันมาก โดยเฉพาะ RAM และ SSD

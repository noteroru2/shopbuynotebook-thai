ถ้าคุณอยากให้ร้านประเมินราคาได้ไวและตรงขึ้น “สเปก” คือข้อมูลที่ช่วยมากที่สุด. บทความนี้รวมวิธีเช็คสเปก Windows แบบง่าย ๆ ไม่ต้องลงโปรแกรมเพิ่ม

## 1) เช็ค CPU / RAM (วิธีเร็ว)

- คลิกขวา Start → System (หรือ Settings → System → About)  
จะเห็น Processor และ Installed RAM

## 2) เช็ค SSD/HDD

- เปิด Task Manager → แท็บ Performance → เลือก Disk  
หรือเปิด “This PC” ดูความจุคร่าว ๆ ได้ (แต่ไม่บอกชนิดดิสก์ชัดเท่า)

## 3) เช็คการ์ดจอ (GPU)

เหมาะกับเครื่องเกมมิ่ง/สายกราฟิก:

- Task Manager → Performance → GPU  
หรือพิมพ์ “dxdiag” ใน Start แล้วดู Display

## 4) ส่งข้อมูลให้ร้านประเมินไว (แนะนำรูปแบบนี้)

- ยี่ห้อ/รุ่น (เช่น Lenovo IdeaPad 5)  
- CPU + RAM + SSD + GPU (ถ้ามี)  
- สภาพ: ใช้งานปกติ/รอย/อาการ  
- อุปกรณ์ที่มี: ที่ชาร์จ กล่อง

จากนั้นไปหน้า [เช็คราคาโน๊ตบุ๊ค](/เช็คราคาโน๊ตบุ๊ค/) หรือ [ตีราคาโน๊ตบุ๊ค](/ตีราคาโน๊ตบุ๊ค/) แล้วส่งข้อมูลได้เลย

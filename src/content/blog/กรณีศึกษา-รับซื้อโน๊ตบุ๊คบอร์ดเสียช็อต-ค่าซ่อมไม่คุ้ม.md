โน๊ตบุ๊คสำหรับเล่นเกม (Gaming Notebook) เป็นคอมพิวเตอร์พกพาที่มีอัตราความร้อนสะสมสูงเป็นพิเศษเนื่องจากใช้ชิปประมวลผล CPU และการ์ดจอแยก GPU ประสิทธิภาพสูง เมื่อพัดลมระบายอากาศเริ่มระบายฝุ่นสะสมไม่ทัน หรือซิลิโคนระบายความร้อนแห้งกรอบ บอร์ดระบบไฟฟ้าก็อาจช็อตถาวรได้ง่ายขึ้น กรณีศึกษานี้เป็นการบันทึกประสบการณ์ตรงในการประเมินราคาและรับซื้อซากโน๊ตบุ๊คเกมมิ่งบอร์ดเสียจากลูกค้าที่ส่งประเมินราคาเพื่อประหยัดต้นทุนและหาทางออกที่เป็นประโยชน์ที่สุด

## ข้อมูลเครื่องและสภาวะความเสียหาย

*   **ยี่ห้อ/รุ่น:** Acer Nitro 5 (โมเดลปี 2022)
*   **สเปกเครื่อง:** CPU Core i7 / RAM 16GB / SSD 512GB / GPU RTX 3060 / จอ 144Hz
*   **อาการเสีย:** ลูกค้าเล่าว่าขณะเล่นเกมหนัก ๆ เครื่องเกิดเสียงดับ "ฟุ่บ" และพัดลมระบายความร้อนหยุดหมุนทันที หลังจากนั้นมีกลิ่นไหม้จาง ๆ เล็ดลอดออกมาจากช่องพอร์ตระบายลมด้านหลัง พยายามเปิดเครื่องใหม่ไฟไม่ติด ไม่มีกระแสไฟตอบสนองพัดลมหรือแป้นพิมพ์
*   **การประเมินจากช่างท้องถิ่น:** ชิ้นส่วนภาคจ่ายไฟหลักช็อตเสียหายลุกลามไปยังชิป GPU การ์ดจอหลัก เสียหลายจุด ค่าซ่อมบอร์ดถูกประเมินไว้ราว 9,500 - 11,000 บาท ซึ่งราคาซื้อขายเครื่องมือสองรุ่นเดียวกันสเปกปกติที่ยังใช้งานได้ดีในตลาดอยู่ราว ๆ 15,000 - 16,000 บาท การซ่อมแซมจึงไม่สมเหตุสมผลในเชิงเศรษฐกิจ

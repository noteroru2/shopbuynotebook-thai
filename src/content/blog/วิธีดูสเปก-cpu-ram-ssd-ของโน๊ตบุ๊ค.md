เวลาคุณอยาก **เช็คราคาโน๊ตบุ๊ค** หรือขอประเมินกับร้านรับซื้อ ข้อมูลที่ทำให้ “ประเมินเร็วและตรง” ที่สุดคือ **สเปกจริงของเครื่อง** โดยเฉพาะ CPU, RAM และ SSD (รวมถึง GPU ถ้าเป็นเกมมิ่ง) เพราะสเปกเหล่านี้มีผลกับราคาตลาดมือสองมากกว่าสีเครื่องหรืออุปกรณ์เสริมบางอย่าง

บทความนี้สรุปวิธีเช็กสเปกบน Windows ให้ครบในไม่กี่นาที และแนะนำรูป/ข้อมูลที่ควรส่งเพื่อประเมินราคาเบื้องต้น

## สเปกที่ควรบอกเวลาส่งประเมินราคา

อย่างน้อยแนะนำให้มี:
- **CPU** (รุ่นเต็ม เช่น i5-1135G7, Ryzen 5 5500U)
- **RAM** (8GB/16GB/32GB)
- **Storage** (SSD/HDD + ความจุ เช่น SSD 512GB)
- **GPU** (ถ้ามีการ์ดจอแยก เช่น RTX 3050/4050)
- รุ่นเครื่อง/ซีรีส์ (เช่น VivoBook, ThinkPad, Nitro)

## วิธีเช็ก CPU/RAM/SSD บน Windows (เลือกวิธีที่ถนัด)

### วิธีที่ 1: Settings → System → About

เหมาะกับผู้ใช้ทั่วไป:
1. Start → Settings
2. System → About

ถ่ายรูปหน้าจอส่วน Processor และ Installed RAM แล้วส่งได้เลย

### วิธีที่ 2: Task Manager (ดู RAM/GPU/Storage ได้ดี)

1. กด `Ctrl + Shift + Esc`
2. ไปที่แท็บ **Performance**

จุดที่ดู:
- CPU (ชื่อรุ่น)
- Memory (ขนาด RAM)
- Disk (ชนิด SSD/HDD และความจุโดยประมาณ)
- GPU (รุ่นการ์ดจอ)

### วิธีที่ 3: System Information (`msinfo32`)

1. กด Windows แล้วพิมพ์ `msinfo32`
2. เปิด System Information

ดู:
- Processor
- Installed Physical Memory
- System Model / SKU (ช่วยยืนยันรุ่นย่อย)

### วิธีที่ 4: ดู Storage แบบชัด (SSD/HDD/ความจุ)

ไปที่ Settings → System → Storage  
หรือถ้าถนัด ให้เปิด “This PC” เพื่อดูความจุไดรฟ์

## ถ้าเป็น Gaming Notebook ต้องส่งอะไรเพิ่ม

- รูปหน้า Task Manager แท็บ GPU (เห็นรุ่น GPU)
- แจ้งว่าเครื่องมีอาการ “ร้อน/พัดลมดัง/ดับเอง” หรือไม่ (ถ้ามีให้บอกตรง ๆ)

## อัปเกรด RAM/SSD มาแล้ว ต้องบอกไหม

ควรบอกครับ เพราะร้านรับซื้อประเมินจาก “สเปกปัจจุบัน” ที่ใช้งานจริง  
เช่นจาก 8GB → 16GB หรือจาก HDD → SSD 512GB อาจช่วยให้ประเมินดีขึ้นได้

## รูปที่ควรส่งคู่กับสเปก

เพื่อให้ประเมินราคาโน๊ตบุ๊คใกล้เคียงขึ้น แนะนำส่ง:
- รูปหน้าเครื่องเปิดจอ
- รูปคีย์บอร์ด
- รูปฝาหลัง/สติ๊กเกอร์รุ่น
- รูปอุปกรณ์ที่มี (ที่ชาร์จ/กล่อง)

ถ้าเครื่องมีอาการ เช่น **เปิดไม่ติด/จอแตก/แบตเสื่อม** ให้ถ่ายรูปหรือวิดีโออาการเพิ่ม

## ทำไม CPU / RAM / SSD ถึงมีผลกับราคา (แบบเข้าใจง่าย)

การประเมินราคาโน๊ตบุ๊คมือสองไม่ใช่การเดา แต่เป็นการเทียบ “ความสามารถใช้งานจริง” กับตลาดมือสองในช่วงนั้น

- **CPU** เปรียบเหมือนสมองของเครื่อง งานทั่วไป กับงานหนักจะต่างกันมาก  
- **RAM** ส่งผลกับการเปิดหลายโปรแกรมพร้อมกัน ถ้าน้อยไปจะหน่วงง่าย  
- **SSD** ส่งผลกับความเร็วโดยรวม เครื่องที่ยังเป็น HDD มักช้ากว่าอย่างชัดเจน  

ดังนั้นการรู้สเปกหลัก ช่วยให้ร้านประเมิน “กลุ่มราคา” ได้เร็วขึ้นก่อนเห็นเครื่องจริง

## วิธีดูสเปกแบบ “เร็วที่สุด” สำหรับคนไม่ถนัดคอม

ถ้าคุณไม่อยากไล่หลายหน้าจอ ให้ทำแค่ 2 อย่างนี้:

1) เปิด **Task Manager** (กด Ctrl + Shift + Esc)  
2) เข้าแท็บ **Performance** แล้วถ่ายรูปหน้า CPU และ Memory และ Disk  

รูป 2 ถึง 3 รูปแบบนี้ มักพอให้ร้านอ่านสเปกได้แล้ว

## Windows: ตัวอย่างจุดที่ควรถ่ายรูป (เพื่อไม่ให้ถามซ้ำ)

แนะนำถ่ายรูป “จุดที่มีข้อมูลครบ” ตามนี้:

- Task Manager → Performance → CPU (เห็นชื่อรุ่น)  
- Task Manager → Performance → Memory (เห็นขนาด RAM)  
- Task Manager → Performance → Disk (เห็นว่าเป็น SSD/HDD และขนาดคร่าว ๆ)  
- ถ้าเป็นเครื่องเกมมิ่ง ให้ถ่ายแท็บ GPU เพิ่ม  

ถ้าคุณถ่ายไม่ทัน ให้ถ่ายวิดีโอสั้น ๆ เลื่อนจาก CPU → Memory → Disk ก็ได้

## ถ้าเครื่องอัปเกรด RAM/SSD มาแล้ว ต้องบอกไหม (ตอบแบบคนขายจริง)

ควรบอกครับ เพราะร้านประเมินจาก “สเปกปัจจุบัน” ที่ใช้งานจริง ไม่ใช่สเปกเดิมตอนซื้อ

ตัวอย่างที่ช่วยให้ประเมินดีขึ้น:
- จาก RAM 8GB → 16GB  
- จาก HDD → SSD  
- จาก SSD 256GB → 512GB หรือมากกว่า  

แต่ทิปสำคัญคือ “บอกตามจริง” และถ่ายรูปหน้า Task Manager ให้เห็น จะทำให้คุยราคาได้ตรงและไม่เสียเวลา

## เช็กให้ละเอียดขึ้น (สำหรับคนอยากให้ประเมิน “ใกล้เคียงจริง” มากขึ้น)

บางเคสสเปกหลักเหมือนกัน แต่ราคาต่างกัน เพราะ “รายละเอียด” เช่น ประเภท SSD หรือจอรีเฟรชเรต หรือการ์ดจอรุ่นย่อย ดังนั้นถ้าคุณสะดวก ลองเพิ่มข้อมูลต่อไปนี้ (ไม่จำเป็นต้องมีครบทุกข้อ)

### 1) SSD เป็น NVMe หรือ SATA (มีผลกับความน่าใช้)

วิธีดูแบบง่าย:

1. เปิด Task Manager → Performance → Disk  
2. บางเครื่องจะบอกคำว่า NVMe หรือ SSD อยู่ในชื่อดิสก์  

ถ้าไม่ขึ้น ให้ใช้วิธีนี้:

1. กด Windows แล้วพิมพ์ `Device Manager`  
2. ไปที่ Disk drives แล้วถ่ายรูปชื่อรุ่น SSD  

แค่ชื่อรุ่น SSD ก็พอให้ร้านอ่านได้ว่าเป็น NVMe หรือ SATA โดยไม่ต้องแกะเครื่อง

### 2) หน้าจอ: ขนาด และรีเฟรชเรต (สำคัญกับเกมมิ่ง)

ถ้าเป็น Gaming Notebook ให้เช็กเพิ่ม:

1. Settings → System → Display → Advanced display  
2. ดู Refresh rate (60Hz / 120Hz / 144Hz / 165Hz ฯลฯ)

ถ่ายรูปหน้าจอนี้ส่งเพิ่ม จะช่วยให้ประเมินใกล้เคียงขึ้น เพราะจอ 144Hz มักมีราคาตลาดต่างจากจอ 60Hz

### 3) การ์ดจอแยก (GPU) รุ่นเต็ม และ VRAM

บางครั้งคนพิมพ์ว่า “RTX” แต่ไม่รู้รุ่นย่อย ซึ่งทำให้ราคาประเมินต่างมาก

วิธีดู:

- Task Manager → Performance → GPU แล้วถ่ายรูป  
- หรือใช้ `dxdiag` แล้วดูแท็บ Display  

### 4) สภาพแบต (เพื่อประเมินตามสภาพจริง)

ถ้าคุณบอกได้ว่าแบต “หมดไว” หรือ “ต้องเสียบตลอด” ร้านจะประเมินตามจริงได้ตรงขึ้น

อ่านต่อ: [โน๊ตบุ๊คแบตเสื่อมขายได้ไหม](/blog/โน๊ตบุ๊คแบตเสื่อมขายได้ไหม/)

## วิธีทำรายงานสเปกแบบไฟล์เดียว (ส่งง่ายมาก)

ถ้าคุณอยากส่งข้อมูลเป็นไฟล์เดียว ลองใช้ `msinfo32`

1. กด Windows แล้วพิมพ์ `msinfo32`  
2. เปิด System Information  
3. ไปที่ File → Export  

แล้วส่งไฟล์หรือรูปหน้าจอส่วน Processor / Installed RAM / System Model ให้ร้านได้เลย

## วิธีเช็กว่า RAM เป็น 8GB หรือ 16GB แบบไม่พลาด

บางคนดูจากสติ๊กเกอร์หน้าบอดี้แล้วไม่ตรงกับปัจจุบัน เพราะอาจมีการอัปเกรด

ให้ยึดค่าที่เห็นใน:

- Task Manager → Performance → Memory  
หรือ  
- Settings → System → About → Installed RAM

ค่าที่นี่คือ “สเปกปัจจุบัน” ที่ใช้ประเมิน

## ถ้าคุณไม่อยากเช็กหลายอย่าง ให้ใช้ “ชุดรูปมาตรฐาน 4 รูป”

เพื่อให้ประเมินราคาไวมาก ให้ส่งรูป 4 อย่างนี้ก่อน:

1) รูปหน้า About (CPU และ RAM)  
2) รูป Task Manager → Disk (SSD/HDD)  
3) รูปสติ๊กเกอร์รุ่นใต้เครื่อง (ยืนยันรุ่น)  
4) รูปหน้าเครื่อง/คีย์บอร์ด (สภาพจริง)  

เท่านี้ส่วนใหญ่ประเมินช่วงราคาได้แล้ว และค่อยถามต่อเฉพาะจุดที่จำเป็น

## สิ่งที่คนมักพลาดเวลา “บอกสเปก” แล้วทำให้ประเมินคลาดเคลื่อน

- ส่งแค่ “ชื่อรุ่นคร่าว ๆ” แต่ไม่ส่ง CPU/RAM/SSD  
- พิมพ์ว่า “i7” แต่ไม่บอกรุ่นย่อย (i7 มีหลายเจนมาก)  
- พิมพ์ว่า “SSD 512” แต่จริงเป็น SSD + HDD เพิ่ม  
- ไม่แจ้งว่าเป็นเครื่องเกมมิ่งที่มี GPU แยก หรือแจ้งผิดรุ่น GPU  
- ไม่แจ้งอาการ (ถ้ามี) เช่น เปิดไม่ติด จอแตก แบตบวม  

การบอกตามจริงและมีรูปประกอบ จะช่วยให้คุณได้คำตอบที่ชัดและเร็วขึ้น

## FAQ เพิ่มเติม (คำถามที่ถามบ่อยจริง)

### เช็กสเปกจากสติ๊กเกอร์บนตัวเครื่องได้ไหม

ได้บ้าง แต่ไม่ชัวร์ เพราะหลายเครื่องถูกอัปเกรด RAM/SSD แล้ว ดังนั้นให้ยึดค่าจาก Windows เป็นหลัก

### ถ้าลง Windows ใหม่ ข้อมูลสเปกจะหายไหม

สเปกไม่หายครับ CPU/RAM/SSD เป็นฮาร์ดแวร์ แต่คุณอาจต้องลงไดรเวอร์ให้ครบเพื่อดูชื่อ GPU ชัดขึ้นในบางกรณี

### ถ้าเครื่องช้า เพราะ HDD ควรบอกไหม

ควรบอกครับ และส่งรูป Disk ใน Task Manager ให้เห็นว่าเป็น HDD หรือ SSD จะช่วยให้ประเมินและแนะนำทางเลือกได้ดีขึ้น

### ถ้าเครื่องมีทั้งการ์ดจอออนบอร์ด และการ์ดจอแยก ต้องบอกตัวไหน

ให้บอก “การ์ดจอแยก” เป็นหลัก และถ่ายรูปหน้า GPU ใน Task Manager จะชัดที่สุด

### เช็กสเปกแล้วจะรู้ราคาทันทีไหม

สเปกช่วยให้ประเมินเร็วขึ้น แต่ราคาจริงยังต้องดูสภาพ และอุปกรณ์ที่มีด้วย เช่น ที่ชาร์จ กล่อง และอาการเสีย (ถ้ามี)

## สรุปแบบใช้ได้ทันที

ถ้าคุณต้องการให้ร้านประเมินราคาไว ให้ยึดหลัก “รูปชัด + ข้อมูลหลักครบ” โดยส่ง CPU RAM SSD และถ้ามี GPU แยกให้ส่งด้วย พร้อมรูปสภาพจริงและที่ชาร์จ ถ้าคุณไม่แน่ใจส่วนไหน ส่งรูปหน้าจอมาได้เลย ร้านช่วยอ่านให้ได้ จากนั้นค่อยคุยต่อเรื่องสภาพ อาการ และนัดหมายตามความสะดวก

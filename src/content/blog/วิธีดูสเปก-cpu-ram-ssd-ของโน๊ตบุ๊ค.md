เมื่อต้องการส่งโน๊ตบุ๊คประเมินราคา หรือ **เช็คราคาโน๊ตบุ๊ค** ก่อนทำการตัดสินใจส่งมอบเครื่อง ข้อมูลที่มีผลต่อการคำนวณราคาประเมินเบื้องต้นมากที่สุดคือ **สเปกฮาร์ดแวร์ภายในของตัวเครื่อง** โดยเฉพาะรุ่นหน่วยประมวลผล (CPU), ขนาดความจุหน่วยความจำ (RAM), ประเภทและขนาดพื้นที่จัดเก็บข้อมูล (SSD/HDD) และรุ่นของชิปการ์ดจอแยก (GPU) เนื่องจากสเปกเหล่านี้เป็นตัวกำหนดมูลค่าการใช้งานจริงในตลาดมือสอง

บทความนี้จะสรุปขั้นตอนและวิธีการตรวจสอบสเปกภายในระบบ Windows ด้วยตัวเองแบบง่าย ๆ เพื่อช่วยให้คุณได้รับราคาเสนอที่รวดเร็วและตรงตามความเป็นจริงที่สุด

> ลิงก์อำนวยความสะดวก: หากต้องการส่งข้อมูลเช็คราคาหรือดูรายละเอียดบริการหลัก สามารถเข้าไปที่หน้า [รับซื้อโน๊ตบุ๊ค](/) หรือหน้าเฉพาะสำหรับผู้ค้นหาคำภาษาอังกฤษที่ [รับซื้อ Notebook](/รับซื้อ-notebook/) และหากต้องการขายเครื่องปกติสามารถตรวจสอบเพิ่มเติมที่หน้า [รับซื้อโน๊ตบุ๊คมือสอง](/รับซื้อโน๊ตบุ๊คมือสอง/)

## 4 องค์ประกอบหลักที่เป็นตัวกำหนดราคาประเมิน

ในการตีราคาโน๊ตบุ๊คมือสอง ช่างจะพิจารณาจากชิ้นส่วนหลักดังนี้:

1.  **หน่วยประมวลผลหลัก (CPU):** ยิ่งเป็นชิปตระกูลใหม่ (เช่น Intel Core i5/i7 เจนเนอเรชั่นล่าสุด หรือ AMD Ryzen 5/7 ซีรีส์ใหม่) จะมีตลาดรองรับกว้างและมีราคากลางที่ดีกว่าชิปรุ่นเก่า
2.  **หน่วยความจำชั่วคราว (RAM):** ขนาดมาตรฐานในปัจจุบันคือ 8GB หากเครื่องของคุณมีขนาด RAM 16GB หรือได้รับการอัปเกรดเพิ่มเองภายหลัง จะช่วยเพิ่มราคาประเมินได้เนื่องจากตอบโจทย์การใช้งานโปรแกรมยุคใหม่
3.  **พื้นที่จัดเก็บข้อมูล (Storage):** โน๊ตบุ๊ครุ่นใหม่ที่ใช้หน่วยความจำความเร็วสูงประเภท **SSD (Solid State Drive)** ขนาดความจุ 512GB หรือ 1TB จะได้รับการประเมินในราคาที่ดีกว่าเครื่องรุ่นเก่าที่ยังใช้ฮาร์ดดิสก์แบบจานหมุน (HDD)
4.  **หน่วยประมวลผลกราฟิก (GPU):** โน๊ตบุ๊คที่มีชิปการ์ดจอแยกประสิทธิภาพสูง (เช่น NVIDIA GeForce RTX Series) จะได้รับการพิจารณาราคาประเมินในกลุ่มเครื่องเล่นเกมหรือทำงานกราฟิกซึ่งเป็นที่ต้องการอย่างมาก

## ขั้นตอนการเช็คสเปกโน๊ตบุ๊ค Windows (เลือกวิธีที่สะดวก)

### วิธีที่ 1: ตรวจสอบผ่านหน้า System About (ง่ายและสะดวกที่สุด)
1. คลิกขวาที่ปุ่ม **Start** (โลโก้ Windows ด้านซ้ายล่าง) แล้วเลือกเมนู **Settings**
2. คลิกเลือกหัวข้อ **System** จากนั้นเลื่อนลงไปคลิกเลือกเมนู **About**
3. ถ่ายรูปหรือบันทึกภาพหน้าจอในส่วนของ **Device specifications** ซึ่งจะแสดงชื่อรุ่น CPU และขนาด RAM อย่างชัดเจน

### วิธีที่ 2: ตรวจสอบผ่านหน้าต่าง Task Manager (ข้อมูลแสดงผลสด)
1. กดปุ่ม `Ctrl + Shift + Esc` บนคีย์บอร์ดพร้อมกันเพื่อเปิดโปรแกรม **Task Manager**
2. คลิกเลือกแท็บ **Performance** (รูปกราฟด้านซ้าย)
3. ในแท็บนี้คุณสามารถคลิกดูข้อมูลแยกตามชิ้นส่วนได้แก่:
   - **CPU:** แสดงชื่อรุ่นเต็มและรหัสซีพียู
   - **Memory:** แสดงขนาดความจุ RAM ปัจจุบัน
   - **Disk:** แสดงประเภทฮาร์ดแวร์ (SSD หรือ HDD) และความจุรวม
   - **GPU:** แสดงโมเดลของการ์ดจอ (หากเครื่องมีการ์ดจอแยก จะแสดงเป็น GPU 1)
4. ถ่ายรูปหรือแคปหน้าจอนี้ในแต่ละหัวข้อเพื่อส่งให้ช่างตรวจสอบ

### วิธีที่ 3: ตรวจสอบผ่าน System Information (สำหรับดูรหัสรุ่นย่อย)
1. กดปุ่ม Windows บนคีย์บอร์ดแล้วพิมพ์คำว่า `msinfo32` ในช่องค้นหา จากนั้นกด Enter
2. หน้าต่างข้อมูลระบบจะแสดงขึ้นมา ถ่ายรูปหน้าแรกนี้ส่งมาทาง LINE เพื่อให้ช่างนำรหัสโมเดลหลัก (System Model) ไปเช็ครายละเอียดสเปกจากโรงงานเพิ่มเติมได้ทันที

## ความปลอดภัยของข้อมูลและการเตรียมเครื่องก่อนส่งมอบ

หลังจากตรวจสอบสเปกเรียบร้อยและตกลงราคากันเรียบร้อยแล้ว ก่อนการส่งมอบเครื่องจริง แนะนำให้ผู้ขายทำการสำรองข้อมูลไฟล์งานที่สำคัญส่วนตัว ย้ายรูปภาพส่วนตัว และออกจากระบบบัญชีต่าง ๆ เช่น Google, Microsoft หรือบัญชีโซเชียลมีเดีย จากนั้นทำการล้างเครื่องคืนค่าโรงงาน (Factory Reset) เพื่อความปลอดภัยและความเป็นส่วนตัวสูงสุด โดยสามารถทำตามขั้นตอนได้ที่บทความ [วิธีล้างข้อมูลก่อนขายโน๊ตบุ๊ค](/blog/ก่อนขายโน๊ตบุ๊คต้องล้างข้อมูลอย่างไร/)

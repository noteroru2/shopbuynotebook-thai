คุณไม่จำเป็นต้องเป็นช่างถ่ายรูปเพื่อขายโน๊ตบุ๊คมือสองครับ แค่รู้ว่า “ร้านอยากเห็นอะไร” ก็มักทำให้การคุยสั้นลงและได้กรอบราคาเบื้องต้น **ใกล้สภาพจริง** มากขึ้น

> กรอบราคาจากรูปยังเป็นแนวทางก่อนตรวจเครื่องจริง — หากสภาพหรือรายละเอียดคนละเรื่องกับที่เห็นในรูป ราคาอาจปรับตามนโยบายร้าน

## ชุดรูปมาตรฐาน (แนะนำ 6–10 รูป)

1. **หน้าจอเปิด** — เห็นขอบจอทั้งสองข้าง ชัดเจนว่ามีเส้น/จุดหรือไม่  
2. **มุมมองคีย์บอร์ด + แทร็คแพด** — สะท้อนสภาพการใช้งาน พื้นผิว  
3. **ฝาเครื่องด้านบน** — เห็นรอยใหญ่/ริ้วคล้ายจากการประกบกระเป๋า  
4. **ด้านข้าง พอร์ตสำคัญ** — ชาร์จ USB-C/เลขกลม เป็นต้น เผื่อเห็นหลวม/สึก  
5. **ใต้เครื่องหรือฉลากรุ่น** — เป็นหลักฐานที่ช่วยให้ชื่อรุ่นตรงกับความเป็นจริง  
6. **ที่ชาร์จของแท้/ที่ใช้จริง** — เห็นสภาพสาย เส้นหัก เป็นต้น  
7. **หน้าสเปก** — เช่น About / System / Task Manager ตามความเข้าถึงของเครื่องคุณ

ถ้ายังไม่แน่ใจว่าถ่ายหน้าจอให้เห็นสเปกจากไหนดี ให้เริ่มจาก [วิธีดูสเปก CPU RAM SSD ของโน๊ตบุ๊ค](/blog/วิธีดูสเปก-cpu-ram-ssd-ของโน๊ตบุ๊ค/)

## ถ้ามีปัญหา “ต้องเน้นรูปเพิ่ม”

- **จอแตก/เป็นเส้น**: ถ่ายมุมที่เห็นเส้นและลองเปลี่ยนมุมมองจากด้านข้าง (บางครั้งเห็นชัดกว่ามองตรง)  
- **เปิดไม่ติด**: ถ่ายเสียบแล้วไฟเข้ายังไง มีไฟสถานะหรือเสียงพัดลมไหม หรือส่งวิดีโอสั้น ๆ  
- **แบตเสื่อม/ชาร์จไม่เข้า**: เพิ่มรายละเอียดจาก [วิธีเช็คสุขภาพแบตโน๊ตบุ๊ค Windows](/blog/วิธีเช็คสุขภาพแบตโน๊ตบุ๊ค-windows/)

ก่อนขายโน๊ตบุ๊ค สิ่งสำคัญไม่แพ้ราคา คือ **ความปลอดภัยของข้อมูลส่วนตัว**. ถ้าล้างข้อมูลไม่ครบ อาจมีไฟล์งาน รูปภาพ หรือบัญชีต่าง ๆ ค้างอยู่. บทความนี้เป็นแนวทาง “ปลอดภัยและทำได้จริง” สำหรับคนทั่วไป

## เช็คลิสต์ก่อนเริ่ม (แนะนำทำทุกครั้ง)

- สำรองไฟล์สำคัญ (External HDD/SSD หรือ Cloud)
- ออกจากบัญชีที่ผูกกับเครื่อง (เช่น Google, Microsoft, LINE, iCloud)
- จดรหัส Wi‑Fi หรือ Serial Software ที่ต้องใช้ต่อ
- ถอดอุปกรณ์เสริมที่ไม่ต้องการขาย (dongle, microSD)

## วิธีล้างข้อมูล Windows (แนวทางง่าย)

1) ไปที่ Settings → System → Recovery  
2) เลือก Reset this PC  
3) เลือก “Remove everything”  
4) หากมีตัวเลือก “Clean data” ให้เลือกเพื่อความปลอดภัยมากขึ้น (ใช้เวลานานกว่า)

หลังรีเซ็ตเสร็จ ให้ทดสอบเปิดเครื่องว่ากลับไปหน้าเริ่มต้น (OOBE) แล้วปิดเครื่องได้เลย

## วิธีล้างข้อมูล MacBook (แนวทางหลัก)

- ออกจาก Apple ID และปิด Find My (สำคัญมาก)  
- ใช้ Erase Assistant (macOS รุ่นใหม่) หรือเข้า Recovery เพื่อลบและติดตั้ง macOS ใหม่  
- ตรวจสอบว่าเครื่องไม่ติด Activation Lock

## หลังล้างข้อมูลแล้ว ทำอะไรต่อ

ถ้าพร้อมขาย แนะนำเช็คราคาเบื้องต้นที่หน้า [เช็คราคาโน๊ตบุ๊ค](/เช็คราคาโน๊ตบุ๊ค/) หรือดูบริการ [รับซื้อโน๊ตบุ๊คมือสอง](/รับซื้อโน๊ตบุ๊คมือสอง/) แล้วส่งรูปประเมินได้ทันที

## สำรองข้อมูลแบบไหนให้ไม่พลาด

ก่อนเริ่มรีเซ็ต แนะนำแบ่งงานเป็น 3 กลุ่ม เพื่อไม่ลืมไฟล์สำคัญ:

- **ไฟล์งาน / เอกสาร**: โฟลเดอร์ Documents, Desktop, Downloads (บางคนเก็บงานไว้ที่นี่)  
- **รูป / วิดีโอ**: Pictures, Photos, โฟลเดอร์ที่ย้ายเอง  
- **ข้อมูลเบราว์เซอร์**: Bookmark, Password manager, 2FA backup codes (ถ้ามี)

ถ้าคุณใช้ Cloud เช่น OneDrive หรือ Google Drive แนะนำเปิดดูอีกครั้งว่าไฟล์สำคัญ “อัปโหลดครบจริง” ไม่ใช่ค้างรอซิงก์

## ออกจากบัญชีอะไรบ้าง (ลดความเสี่ยงเรื่องบัญชีค้าง)

หลายเคสที่คนล้างข้อมูลแล้ว “ยังไม่จบ” มักเกิดจากบัญชีที่ผูกกับเครื่อง โดยเฉพาะ:

- **บัญชี Microsoft / Windows**: ถ้าใช้ Microsoft account ให้ Sign out และตรวจที่ Settings ว่าไม่เหลือบัญชีของคุณ  
- **Google account**: Gmail/Chrome sync อาจค้างอยู่ ให้ Sign out ก่อน  
- **LINE / Facebook / แอปแชท**: ถ้ามีแอปในเครื่อง แนะนำออกจากระบบ  
- **MacBook**: Apple ID และ Find My ต้องปิดก่อนเสมอ (เพื่อไม่ติด Activation Lock)

แนวคิดง่าย ๆ คือ “อะไรก็ตามที่ทำให้คนอื่นเปิดเครื่องแล้วเข้าถึงข้อมูลคุณได้” ต้องออกให้หมดก่อน

## Windows: รีเซ็ตแบบไหนถึงจะปลอดภัยขึ้น

ใน Windows วิธีมาตรฐานคือ Reset this PC และเลือก “Remove everything” แต่ถ้าเครื่องมีข้อมูลสำคัญมาก แนะนำดูตัวเลือกเพิ่มดังนี้:

- **Cloud download vs Local reinstall**: โดยทั่วไปทำได้ทั้งคู่ ถ้าเน็ตเร็ว Cloud download จะได้ไฟล์ระบบใหม่  
- **Clean data**: ถ้ามีตัวเลือกนี้ ให้เปิดเพื่อให้ลบข้อมูลละเอียดขึ้น (ใช้เวลานานกว่า)

หลังจากรีเซ็ตเสร็จ ให้สังเกตว่าหน้าแรกที่ขึ้นคือหน้า “ตั้งค่าเครื่องใหม่” และไม่มีชื่อบัญชีคุณปรากฏอยู่

### เช็กจุดเสี่ยงที่คนมักลืมใน Windows

- ลืมลบข้อมูลในไดรฟ์อื่น (ถ้ามีหลายไดรฟ์)  
- ลืม Sign out แอปแชท/บราวเซอร์ก่อนรีเซ็ต  
- ลืมสำรองไฟล์ใน Downloads / Desktop  
- ลืมถอด MicroSD / USB Dongle ที่ไม่ได้ตั้งใจขาย

## MacBook: ทำไมต้องออก Apple ID และปิด Find My ก่อน

MacBook มีระบบ Activation Lock ถ้า Find My ยังเปิดอยู่ ต่อให้ลบข้อมูลไปแล้ว เครื่องอาจยังถูกล็อกและใช้งานต่อไม่ได้
ดังนั้น “ออก Apple ID + ปิด Find My” เป็นขั้นตอนที่สำคัญที่สุดก่อน Erase/ติดตั้งใหม่

## ก่อนเริ่ม: 3 จุดที่คนมักพลาด แล้วทำให้ “ล้างแล้วไม่จบ”

### 1) สำรองข้อมูลไม่ครบ

หลายคนสำรองแค่ Documents แต่ลืม Desktop และ Downloads ซึ่งเป็นที่เก็บไฟล์งานแบบไม่ตั้งใจ

### 2) ลืมออกจากบัญชีบางตัว

เช่น Chrome Sync, LINE, Microsoft Office, หรือแอปธนาคาร/แอปงาน ถ้ายังล็อกอินค้างอยู่ มีโอกาสทิ้งข้อมูลไว้

### 3) ลืมไดรฟ์อื่น หรืออุปกรณ์เสริม

บางเครื่องมีหลายไดรฟ์ หรือเสียบ MicroSD/USB ค้างไว้ ทำให้ข้อมูลยังอยู่แม้รีเซ็ตบางแบบ

## สำรองข้อมูลแบบเป็นระบบ (ทำครั้งเดียวแล้วไม่ลืม)

ให้แบ่งไฟล์เป็น 4 กลุ่ม แล้วไล่เช็กทีละกลุ่ม:

1) **งาน/เอกสาร**: Documents, Desktop, Downloads, โฟลเดอร์โปรเจกต์  
2) **รูป/วิดีโอ**: Pictures, Photos, โฟลเดอร์ที่ย้ายเอง  
3) **เบราว์เซอร์**: Bookmark, Password manager, ไฟล์ 2FA backup codes  
4) **งานเฉพาะทาง**: ไฟล์โปรแกรมบัญชี งานกราฟิก งานตัดต่อ

ถ้าคุณใช้ Cloud (OneDrive/Google Drive) ให้เปิดดูว่า “ซิงก์เสร็จจริง” ไม่ใช่ค้างอัปโหลด

## เช็กลิสต์ “ออกจากระบบ” ที่ควรทำก่อนล้าง

รายการนี้ช่วยลดความเสี่ยงข้อมูลค้างและช่วยให้เครื่องพร้อมส่งต่อ

### Windows / ทั่วไป

- Google account (Gmail/Chrome)  
- Microsoft account (Windows/OneDrive/Office)  
- LINE / Facebook / แอปแชท  
- โปรแกรมจัดการรหัสผ่าน (ถ้าคุณใช้)  
- โปรแกรมงานที่มี License ผูกเครื่อง (ถ้ามี)  

### MacBook (สำคัญเป็นพิเศษ)

- Apple ID / iCloud  
- Find My ต้องปิดก่อนเสมอ (ไม่งั้นติด Activation Lock)  

อ่านต่อแบบละเอียด: [ขาย MacBook ต้องเตรียมอะไรบ้าง](/blog/ขาย-macbook-ต้องเตรียมอะไรบ้าง/)

## Windows: รีเซ็ตแบบ “ลบทุกอย่าง” ทีละขั้นตอน (ทำตามได้)

เหมาะกับคนที่ต้องการล้างข้อมูลแบบมาตรฐาน

1) ไปที่ Settings → System → Recovery  
2) เลือก Reset this PC  
3) เลือก Remove everything  
4) ถ้ามีตัวเลือก Clean data ให้เลือก (ปลอดภัยขึ้น แต่ใช้เวลานานกว่า)  
5) เลือก Cloud download หรือ Local reinstall ตามความสะดวก  

เมื่อทำเสร็จ เปิดเครื่องเช็กว่าเข้าไปหน้า “ตั้งค่าเครื่องใหม่” (OOBE) แล้วปิดเครื่องได้เลย

### Windows: ถ้ามีหลายไดรฟ์ ต้องระวังอะไร

ถ้าเครื่องคุณมีไดรฟ์มากกว่า 1 ลูก ให้ดูตัวเลือกตอน Reset ว่าลบเฉพาะไดรฟ์ระบบ หรือ ลบทุกไดรฟ์

ถ้าคุณไม่แน่ใจ ให้ถ่ายรูปหน้าจอตัวเลือก แล้วถามก่อนทำจริงก็ได้ เพื่อกันพลาด

## MacBook: ล้างข้อมูลอย่างปลอดภัย (สรุปให้ทำจริงได้)

หลักสำคัญมี 2 อย่าง:

1) **ออก Apple ID และปิด Find My**  
2) **Erase/ติดตั้ง macOS ใหม่** ให้เครื่องกลับไปหน้าเริ่มต้น

### วิธีที่ง่ายที่สุด (macOS รุ่นใหม่): Erase All Content and Settings

1) System Settings → General → Transfer or Reset  
2) เลือก Erase All Content and Settings  
3) ทำตามขั้นตอนจนจบ

วิธีนี้เหมาะกับคนทั่วไป และลดโอกาสพลาดเรื่องบัญชี

### วิธี Recovery (ใช้ได้กับหลายรุ่น)

- Intel: กด Command + R ตอนเปิดเครื่อง  
- Apple Silicon: กด Power ค้างจนเห็น Options แล้วเข้า Recovery  

จากนั้นใช้ Disk Utility ลบไดรฟ์ และติดตั้ง macOS ใหม่

## หลังล้างข้อมูล: ต้องเช็กอะไรอีก 5 อย่างก่อนส่งมอบ

1) เครื่องเข้าได้แค่หน้า “ตั้งค่าเริ่มต้น” ไม่มีชื่อบัญชีคุณ  
2) ไม่มีไฟล์ส่วนตัวค้างใน Desktop/Documents  
3) ถอด MicroSD/USB/Dongle ที่ไม่ตั้งใจขาย  
4) เก็บที่ชาร์จให้ครบ และตรวจว่าชาร์จได้  
5) ถ่ายรูปสภาพจริง 3 ถึง 5 รูป เพื่อขอประเมินราคา

## ข้อควรระวัง “ข้อมูลในรูป” ตอนส่งประเมินราคา

เวลาแอดไลน์ส่งรูปเพื่อประเมิน คุณอาจเผลอส่งข้อมูลที่ไม่จำเป็น เช่น:

- สติ๊กเกอร์ที่มี Serial ชัดมาก  
- ภาพหน้าจอที่มีอีเมล ชื่อผู้ใช้ หรือข้อมูลลูกค้า  
- รูปไฟล์งานบนเดสก์ท็อปที่อ่านชื่อไฟล์ได้  

ทางแก้คือ:

- ถ่ายรูปเฉพาะส่วนที่จำเป็น  
- ถ้าเป็นหน้าจอ ให้ซูมเฉพาะสเปก  
- ถ้ากังวล ให้ปิดท้าย Serial 2 ถึง 4 ตัว

## FAQ เพิ่มเติม (คำถามที่คนถามบ่อย)

### 1) รีเซ็ตอย่างเดียวพอไหม

ส่วนใหญ่พอครับ ถ้าทำถูกขั้นตอนและเลือก Remove everything หรือ Erase ตามมาตรฐาน แต่ถ้าคุณมีข้อมูลอ่อนไหวมาก แนะนำใช้ตัวเลือก Clean data (Windows) หรือทำ Erase ตามขั้นตอนที่ถูกต้อง

### 2) ล้างแล้วจะทำให้ราคาเพิ่มไหม

การล้างข้อมูลไม่ได้ทำให้สเปกเพิ่ม แต่ช่วยให้ “ปิดงานไว” และลดความกังวลเรื่องข้อมูลส่วนตัว ทำให้คุณตัดสินใจขายได้ง่ายขึ้น และร้านตรวจสภาพได้เร็วขึ้น

### 3) เครื่องเปิดไม่ติด ยังล้างข้อมูลได้ไหม

ถ้าเปิดไม่ติด อาจล้างเองไม่ได้ แต่ยังขายตามสภาพได้ครับ แนะนำส่งรูปสติ๊กเกอร์รุ่นใต้เครื่อง + วิดีโออาการ แล้วให้ร้านประเมินตามสภาพ

อ่านต่อ: [โน๊ตบุ๊คเปิดไม่ติดขายได้ไหม](/blog/โน๊ตบุ๊คเปิดไม่ติดขายได้ไหม/)

### 4) ถ้ายังไม่พร้อมล้าง ขอประเมินก่อนได้ไหม

ได้ครับ ขอประเมินเบื้องต้นก่อน แล้วค่อยล้างก่อนส่งมอบจริงก็ได้

## ถ้าคุณกังวลมากเรื่องข้อมูล: ทำให้ “ปลอดภัยขึ้น” อีกระดับ

สำหรับคนที่มีข้อมูลสำคัญมาก (งานลูกค้า เอกสารสำคัญ หรือรูปส่วนตัวจำนวนมาก) คุณสามารถเพิ่มความปลอดภัยได้ด้วยแนวทางนี้

### Windows: ตัวเลือก Clean data และการเข้ารหัส

- ถ้าตอน Reset มีตัวเลือก **Clean data** ให้เลือก เพราะจะลบละเอียดขึ้น (แต่ใช้เวลานานกว่า)  
- ถ้าคุณเคยเปิด **BitLocker** ไว้ (บางเครื่องเปิดอัตโนมัติ) การเข้ารหัสช่วยลดความเสี่ยงข้อมูลถูกกู้คืนได้อยู่แล้ว แต่ก็ยังควร Reset แบบ Remove everything เพื่อให้เครื่องพร้อมขาย

### MacBook: FileVault และการ Erase ตามขั้นตอน

- ถ้าคุณเปิด **FileVault** ไว้ ข้อมูลบนดิสก์จะถูกเข้ารหัส ช่วยเพิ่มความปลอดภัย  
- แต่ยังต้องทำ Erase/ติดตั้ง macOS ใหม่ และต้องปิด Find My ให้เรียบร้อย เพื่อไม่ให้ติดล็อก

แนวคิดสำคัญคือ “เข้ารหัสช่วยกันกู้ข้อมูล” แต่ “การรีเซ็ต/Erase” ช่วยทำให้เครื่องพร้อมส่งต่อและไม่เหลือบัญชีค้าง

## หลังล้างแล้ว: เช็กบัญชีออนไลน์ 3 อย่าง (เผื่อเคยผูกเครื่องไว้)

บางคนล้างเครื่องแล้ว แต่ลืมว่ามีบัญชีบางอย่างที่ผูกอุปกรณ์ไว้

แนะนำเช็ก:

1) **Apple ID / iCloud**: ดูรายชื่ออุปกรณ์ที่ผูกอยู่ และลบเครื่องที่ขายออก  
2) **Microsoft account**: ดู Devices ที่ผูกกับบัญชี (ถ้าเคยใช้)  
3) **Google account**: ดู Security → Your devices แล้ว Sign out อุปกรณ์เก่าถ้าจำเป็น  

ทำ 3 อย่างนี้ช่วยลดความเสี่ยงเรื่องอุปกรณ์ค้าง และทำให้คุณสบายใจขึ้น
